admiralty-dist

Distribution channel for the Admiralty Fleet Command System. Wheels only — no source code in this repository.

Install

pip install admiralty --extra-index-url https://tavisbasing.github.io/admiralty-dist/

After installation:

adm version
adm licence activate <YOUR-KEY>
adm quickstart
You don't need a GitHub account to install. The documentation covers activation, the licence flow, and the quickstart demo.

What's here

  • /simple/ — PEP 503 index (this is what pip reads)
  • /simple/admiralty/ — per-package index with SHA-256 hashes
  • /wheels/ — built wheels (.whl)
  • Releases — every published version with notes + SHA-256

Verifying a download

Each release page lists the wheel's SHA-256 hash. For production environments use pip --require-hashes with a pinned requirements.txt.
